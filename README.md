# 🌤 Telegram Weather Notifier Bot

Асинхронный Telegram-бот на **Python (aiogram v3)**, который отправляет ежедневные уведомления о погоде в выбранном городе в заданное пользователем время.

Бот позволяет настроить город, время уведомления и таймзону, а затем автоматически присылает актуальную погоду каждый день.

---

## 🚀 Возможности

- 📍 Выбор города (с поддержкой нескольких совпадений)
- ⏰ Настройка времени ежедневного уведомления
- 🕒 Поддержка пользовательской таймзоны
- 🔔 Включение / выключение уведомлений
- 🌤 Просмотр текущей погоды по кнопке
- 💾 Хранение данных в SQLite
- ⚡ Полностью асинхронная архитектура

---

## 🛠 Используемые технологии

- Python 3.10+
- aiogram v3
- aiohttp
- aiosqlite
- Open-Meteo API (без API-ключа)
- python-dotenv

---

## 📦 Архитектура проекта

- `main.py` — основной файл бота  
- SQLite — хранение настроек пользователей  
- Async scheduler — фоновая отправка уведомлений  
- Open-Meteo API — получение погодных данных  

Проект реализован без сторонних планировщиков (APScheduler и т.д.), что делает архитектуру простой и прозрачной.
